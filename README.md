# UITest_Museum
Este repositorio contiene el resultado de un ejercicio de diseño del primer boceto la Interfaz Grafica de una aplicación móvil para el museo nacional de Colombia
y es parte del ejercicio de la materia de INGENIERÍA WEB I del pregrado de Ingenieria de software de la Universidad Manuela Beltran
